# **App Name**: Castaway

## Core Features:

- Feed Display: Display a feed of 'casts' from users the user follows.
- Cast Creation: Allow users to create and post 'casts' (short text updates).
- User Following: Enable users to follow other users to see their casts in the feed.
- User Profiles: Display basic user profiles with username and casts.
- Notifications: Inform users of new followers or interactions (likes, recasts) with their casts.

## Style Guidelines:

- Primary color: Deep purple (#6C2CBD), for a modern, distinct, eye-catching look that takes inspiration from the purple highlights used in the example user interface. 
- Background color: Very light grayish-purple (#F4F0F9), creating a soft and clean backdrop.
- Accent color: Blue-purple (#505CDD) to complement the purple, draw attention, and convey the app's modern vibe.
- Body and headline font: 'Inter' sans-serif for a modern, neutral, and readable style.
- Use simple, outlined icons for navigation and actions, maintaining a clean interface.
- Layout inspired by Farcaster UI, single column feed, and sidebar navigation
- Subtle transitions on hover and click events for a polished user experience.
